This project presents an interactive Power BI dashboard built using the Blinkit sales dataset (8,523 rows). The dashboard provides insights into product performance, outlet performance, customer preferences, and sales trends.

Key Features

Analysis of Sales by Item Type, Item Fat Content, and Outlet Type

Outlet performance comparison by size, location, and establishment year

Item visibility vs. sales relationship

Filtering based on fat content, item type, and outlet category

Cleaned and transformed data with standardized categories

Interactive visuals with slicers and KPIs

Tools Used

Power BI

Excel/CSV dataset

Power Query (data cleaning)

DAX Measures
